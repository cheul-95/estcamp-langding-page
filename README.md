## 💻 로컬에서 실행하기

1. 이 저장소를 클론하거나 zip으로 다운로드합니다.
2. `index.html` 파일을 브라우저에서 직접 열면 됩니다.

```
git clone https://github.com/cheul-95/estcamp-langding-page.git
cd estcamp-langding-page
start index.html
```
